## Cómo actualizar las fotos

1. Guardá la foto nueva en la carpeta `img/`
2. Nombrala igual que la que querés reemplazar (ej: `trabajo1.jpg`)
3. Subí el cambio a GitHub — Vercel actualiza la web automáticamente

## Cómo actualizar textos

Abrí `index.html` con el bloc de notas y buscá el texto que querés cambiar.

## Deploy

El sitio está publicado en **Vercel** y se actualiza solo cada vez que se sube un cambio a GitHub.
